# 麻袋的产品打怪日记 - 个人作品集网站

这是一个产品经理的个人作品集网站，包含教育背景、项目经验、兴趣爱好和AI聊天机器人功能。

## 本地运行

**环境要求:** Node.js

1. 安装依赖:
   `npm install`
2. 在 [.env.local](.env.local) 中设置 `ZHIPUAI_API_KEY` 为你的智谱AI API密钥
3. 运行应用:
   `npm run dev`

## 部署

1. 构建项目:
   `npm run build`
2. 部署到 GitHub Pages:
   `npm run deploy`

## 技术栈

- React 18
- TypeScript
- Vite
- Tailwind CSS
- 智谱AI (GLM-4)

## 功能特性

- 响应式设计
- AI聊天助手
- 项目展示
- 教育背景介绍
- 兴趣爱好分享
